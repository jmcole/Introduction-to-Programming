Telling Computers What To Do

Introduction to Serious Programming

Computer

A computer is a machine that can do almost anything. However, it must be given specific instructions on how to accomplish its task.

Program

A program is a specific sequence of instructions that tell a computer what to do.

Program Language

A program languages are used by programmers to write programs. Languages can be interpreted or compiled. Python is interpreted, which means it uses a seperate program to convert the code to code the computer can understand. Compiled language, such as C++, create an executable file that does not need a compiler to break the program down so the computer can understand it.

Python

Python is an interpreted language. It is named after Monty Python.

Grammar

Programming languages have a specific grammar just like human or natural languages. A computer can only read code written in perfect grammar, other wise it will not run.

Python Expressions

In Python grammar, an expression is print 4 + 5. An example of invalid syntax would be print 2 +. In Python it must match the grammar exactly.

Recursive Definition

A recursive definition requires one rule where we can keep going and one where we can stop.

Find Method

Functions or "Procedures"

A function is something that takes input, does something to that input, and then produces output. The value of a function is that it allows programmers to do reuse code, because once it is defined you don't have to define it again. To create a function in Python you use the following method. 
            
              def square(x): #use keyword def, followed by name, and then parentheses around the parameters
              answer = x * x
              return answer

              # to call the function, write the name and give it values
              print square(4)
              >>>16
The return keyword is vital to the function because it tells Python what it should produce as an output.

"while Loops"

A loop allows a program to run over and over until it gets a result. It gets a result when the test condition is met. break allows a loop to stop, even when the test condition is not met. 
            
              def print_numbers(numbers):
                  i = 1
                  while i <= numbers:  #i <= numbers is the test condition
                      print i
                      i = i + 1

              print_numbers(3)
              #>>> 1
              #>>> 2
              #>>> 3

              def print_numbers(numbers):
                  i = 1
                  while i <= numbers:
                      if i == 2:
                          break
                      print i
                      i = i + 1

The built-in method index gives the position of the first match within a list of a particular element. 
            
              def find_element(list,value):
                for e in list:
                    if e == value:
                        return list.index(e)
                return -1


                print find_element([1,2,3],3)
                #>>> 2

                print find_element(['alpha','beta'],'gamma')
                #>>> -1
