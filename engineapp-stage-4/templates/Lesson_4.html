HTML

HTTP Request

When a browser requests a website, the browser sends a get request along with headers to the server. The get request is comprised of three parts.
GET/FOO HTTP/1.1-request line
  1. GET-is the method, POST will also be used in the class
  2. /foo-path used for making the request
  3. http/1.1-version 1.0 will also be discussed
GET is the most common method for requesting information from a server.
Headers are also sent along with the request.
  • Headers are in the form name:value
  • host:www.example.com from the url. Required in 1.1. Servers may have multiple names
  • user-agent: describes who is making the request, generally the browser

HTTP Response

After receiving a request the server will respond back to the user. http/1.1 200 ok - This is a status line.
  1. http/1.1 - version
  2. 200 - status
  3. ok - Reason phrase
Common staus codes
  1. 200 = OK
  2. 302 = found
  3. 404 = not found (browser side error)
  4. 500 = Server error (server side error)
Headers are also included in the response.
  • Date:tue march 2012 04:33:33 GMT(when request happened)
  • Server:Apache 2.2.3(not required and not recomended)

String Substitution

%s is used in Python to substitute a variable into a string 
              
                given_string = "I think %s is a perfectly normal thing to do in public."
                def sub1(s):
                  return given_string % s

                print sub1("running")
                # => "I think running is a perfectly normal thing to do in public."
String substitution can also be done with multiple variables 
              
                def sub2(s1, s2):
                  return given_string2 % (s1,s2)

                print sub2("running", "sleeping")
                # => "I think running and sleeping are perfectly normal things to do in public."
String substitution can also be done with a dictionary. 
              
                given_string2 = "I'm %(nickname)s. My real name is %(name)s, but my friends call me %(nickname)s."
                def sub_m(name, nickname):
                    return given_string2 % {"name":name,"nickname":nickname}

                print sub_m("Mike", "Goose")
                # => "I'm Goose. My real name is Mike, but my friends call me Goose."
