Webpages, Documents, and Structure

The Basics of the Web and HTML

How the Web Works

Hyper Text Markup Language is the language of the internet. HTML is interpreted by a computer's browser. These documents reside on servers. When a user accesses a webpage their computer sends an HTTP request to the server. The server finds the document and sends it across the internet where it is interpreted by the user's browser. The web is a collection of HTML documents. There are approximately 30 billion pages on the World Wide Web.

HTML Makeup

  • Text Content- "What you see"
  • Markup- "What it looks like" references to other documents e.g. images and video and links to other pages.
  • Element-An element is comprised of an opening tag, content, and a closing tag.

HTML Attributes

  • Tag Attribute< tag attribute="value" > contents</tag>
  • Anchor Tag <a>- creates a link
  • Creating a link <a href="www.reddit.com" >derp</a> derp
  • image tag- <img><img src="url of image" alt="text">

White Space

  • HTML renders all white space as a single space
  • <br> break (void tag does not need close)
  • <p> paragraph tag</p>(Requires Close)

Inline vs. Block

  • <br>tag is inline
  • <p> tag is a block element. It creates an "invisible box" around the content.
  • <span></span>- inline
  • <div></div>- block

HTML Documents

  • Head- contains title and metadata, CSS, and Javascript
  • Body- contains content

Creating a Structured Document with HTML

Tree Structure of HTML

  • HTML has a tree like structure, wth elements nested inside other elements. Link to Wikipedia article on Tree Structure. When programmers talk about the DOM(Document Object Model) they are talking about the tree like structure of a web page.
  • HTML controls the "structure" of the web page
  • CSS controls the "style" of a page (How it looks)
  • CSS helps web designers avoid repetition. Avoiding repetition helps to keep code "clean" and more easily modifiable.
  • Boxes- webpages are composed of "boxes". <div> tag defines boxes. Class attributes are the labels on boxes.

Avoiding Repetition

  • Avoiding errors-The less you do it, the less chance you have of making errors.
  • Consistency-Reducing repetition makes pages more consistent and easier to change.

Adding CSS Style to HTML Structure

Understanding CSS

CSS is code written to control the "style" of HTML elements. 

          h1 {
          background-color : black;
          color: white;
          }
h1 is a selector. This tells the browser what to apply the rules too. The rules are between the {} braces.background-color : black is a rule. background-color is an attribute and black is the value of that attribute.

To add style to specific h1 or other element use class selectors. Class selectors are written in the format .class-name

Adding CSS to a webpage

CSS can be added in three ways.
  1. Write CSS in the <head> of your HTML. Best for small projects and testing.
  2. Link your HTML to a separate CSS file. Most organized method and best for larger projects.
    1. Write all of your structural HTML in one file (let's call it main.html).
    2. In a separate file (let's call it main.css), put all of your CSS code.
    3. Add a <head> element to the top of your HTML.
    4. Add a <link> tag inside the <head> element. Since <link> is a "void tag" you don't need to add a closing </link>.
    5. Add the following attributes to your <link rel="stylesheet" type="text/css" href="main.css">
  3. Write your style inline with your HTML- This results in a lot of repeated code.

Box Sizing

HTML is comprised of boxes. Boxes have 4 components and because of this it can be hard to get sizing correct. There are two techniques to get sizing correct.
  1. Set sizes in terms of percent rather than pixels.
  2. Set the box-sizing attribute to border-box for every element.

Box Positioning

  1. Divs are block elements (as opposed to inline), so by default they take up the entire width of a page.
  2. Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other.
